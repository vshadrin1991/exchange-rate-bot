# exchange-rate-bot

Telegram exchange rate bot

# prepare data

1. Navigate to configs/config.py
2. Set up the following data
   | Key | Description |
   | ----------------------------------- | ------------------------------------ |
   | **TELEBOT_TOKEN** | your bot token in the telegram app |
   | **EXCHANGE_API** | url excange api|

# start commands

**poetry run python3 main.py**
